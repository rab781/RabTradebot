## 🚀 Quick Start Instructions

### **1. Configure Telegram Bot**

Add your bot token to `config/config.json`:

```json
{
  "telegram": {
    "enabled": true,
    "token": "YOUR_BOT_TOKEN"
  }
}
```

### **2. Start the Bot**

```bash
# Development mode (recommended for testing)
npm run dev:phase2

# Production mode
npm run build:phase2
npm run phase2

# Quick build and run
npm run quick:phase2
```

### **3. Test AI Commands**

Send these commands to your bot:

```
/mlpredict BTC      # Complete AI analysis
/system             # Verify all services running
/help               # See all available commands
```
